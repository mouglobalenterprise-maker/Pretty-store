# Pretty Store — Frontend Animation Experiment

A single-page salon/storefront concept built purely as a frontend playground — used to experiment with cinematic animation, 3D-style visual effects, and modern UI techniques outside of the backend-heavy systems that make up most of my other repositories.

This is not a client deliverable or a finished product — it's a sandbox for testing layout, motion, and visual polish techniques (animated transitions, depth/glassmorphism effects, large typography) that I plan to bring into future production frontend work.

## Tech Stack

HTML, CSS, JavaScript — no frameworks, no build step.
